# MFTA Credit Risk Application

This project demonstrates a credit risk prediction workflow using machine learning within a fintech context. The goal is to evaluate borrower risk, interpret model outputs, and connect predictions to real-world financial decision-making.

## Objective

* Predict probability of default (PD) for loan applicants
* Translate model outputs into actionable lending decisions
* Evaluate model performance and limitations

## Approach

* Data preprocessing and feature engineering
* Model development (e.g., logistic regression, tree-based models)
* Model evaluation (accuracy, precision/recall, ROC-AUC)
* Interpretation of results (feature importance, risk segmentation)

## Tech Stack

* Python (pandas, scikit-learn)
* Jupyter Notebook
* (Optional) FastAPI or Flask for deployment
* (Optional) Front-end dashboard

## Key Concepts

* Credit risk modeling
* Classification problems in finance
* Model evaluation and trade-offs
* Bias, fairness, and model transparency
